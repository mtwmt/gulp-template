# gulp-template

已更新至 gulp4 請先確認 

`node 8.11.1` 與 `npm 5.6.0` 以上版本

### 安裝pageage套件
```
$ npm install
```

### gulp-cli全域
```
$ npm install gulp-cli -g
```

npm run
```
$ gulp
```
---

### gulp-cli區域
```
$ npm install gulp-cli
```
在package.json 加寫一串執行指令
```
"scripts": {
  "watch": "gulp"
},
```
npm run
```
$ 
```
